회원가입 정보 : 구체적으로 어떤 정보를 저장할 것인지, 관련 사이트와 동일해도 상관 없음
메인 페이지 메뉴(nav) 구성 : 네비게이션에 들어갈 구체적인 항목 전체, 배치할 위치 및 모양(header 밑에 가로로 배치할 것인지, 왼쪽에 세로로 배치할 것인지, 항목 선택 시 서브 메뉴를 보이게 할 것인지 아니면 바로 그 항목에 해당하는 페이지로 이동할 것인지)
회사 게시판 : 네비게이션 항목을 클릭했을 때 보여질 게시판 디자인, 게시판에 글쓰기를 했을 때 입력해야할 항목 구체적으로, UI 배치
회사 컨텐츠 게시판 : 사용자가 글을 쓰는 공간이 아닌 회사에서 제공하는 or 홍보하는 컨텐츠의 게시판의 디자인 및 컨텐츠 예시 3개, 기존의 글 목록만 보이는 게시판이 될 수도 있고, 썸네일(간단한 그림)을 포함하는 또는 썸네일만 보이는 게시판이 될 수도 있음

월드비전 예시

1. 회원가입 정보
STEP 01 가입여부 확인
 - 구분(개인, 기업/교회, 단체)
 - 이름
 - 생년월일 : 입력이 아니라 옵션 박스로 선택
 - 휴대전화 : 앞 3자리 옵션 박스, 나머지 직접입력 
STEP 02 회원 종류 선택
 - 일반회원가입(버튼)
STEP 03 회원 정보 입력
 - 아이디(중복확인)
 - 비밀번호
 - 비밀번호 확인
 - 이름(이전에 입력한 정보 그대로 출력)
 - 성별(체크박스)
 - 휴대전화(이전에 입력한 정보 그대로 출력)
 - 생년월일(이전에 입력한 정보 그대로 출력)
 - 이메일(ID는 직접 입력 @이후 도메인은 옵션박스로 선택 or 직접 입력)
 - 약관/개인정보 수집 및 이용 관련 동의(필수 : 이용약관, 개인정보 수집 및 이용제공에 대한 동의, 선택 : 월드비전 소식 및 후원/서비스 안내를 위한 개인정보 수집 및 이용 동의)


2. 메인 페이지 메뉴(nav) 구성
1) 오른쪽 상단 서브 메뉴
 - 월드비전, 참여하기, 자료센터, 마이월드비전, 로그인, 회원가입(로그인 시 보이지 않음)
2) 메인 페이지 메인 메뉴
 - 왼쪽 상단에 월드비전 로고가 있고 그 오른쪽에 차례로 메뉴 항목 배치
 - 후원하기, 사업소개, 스토리

3. 회사 게시판
 - 필수 구현 기능에 해당하는 게시글 검색 및 파일 업로드가 같이 되는 게시판은 없음
 - 파일 업로드 구현을 위해서는 온라인 상담에서 1:1 상담하기 게시판을 생성해야함
 - 게시글 검색을 위한 게시판이 필요한데 이것은 회사 컨텐츠 게시판에서 구현을 해도 될 듯

4. 회사 컨텐츠 게시판
 - 월드비전 메인 페이지에서 볼 수 있는 World Vision News를 구현, 메인 페이지에서 +를 누르면 게시판 페이지로 이동, 관련 컨텐츠는 그대로 복사
